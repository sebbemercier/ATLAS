# Recommended Datasets for ATLAS (Inventory & Knowledge)

- **Spider (Text-to-SQL)**: Comprehensive cross-domain dataset for complex query generation. [Hugging Face](https://huggingface.co/datasets/spider)
- **Brazilian E-Commerce Public Dataset (Olist)**: 100k orders for real-world schema mapping. [Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- **WikiSQL**: Large-scale dataset for simpler NL-to-SQL tasks. [Salesforce](https://github.com/salesforce/WikiSQL)
